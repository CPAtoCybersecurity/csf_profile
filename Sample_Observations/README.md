# Sample Observations

This folder contains example assessment observations documenting findings for CSF subcategories.

## What Goes Here

Documented observations from control assessments. Each file should include:

- **CSF Subcategory** - The specific control assessed
- **Assessment Period** - Which quarter/period this observation covers
- **Observation** - What was found during testing
- **Testing Methods** - Examine, Interview, and/or Test
- **Scores** - Actual vs. Target scores
- **Gap Analysis** - Identified deficiencies (if any)
- **Linked Artifacts** - Supporting evidence

## File Naming Convention

```
{CSF-ID}-{Quarter}-Observation.md

Examples:
- DE-AE-02-Q1-Observation.md
- GV-SC-04-Q1-Observation.md
- GV-OV-01-Q2-Observation.md
```

## Template

See any existing file in this folder for the standard format.

## Contributing

1. Pick an unassigned CSF subcategory from the [Issues](https://github.com/CPAtoCybersecurity/csf_profile/issues)
2. Create your observation following the naming convention
3. Submit a Pull Request referencing the issue number
