# ID.AM-01: Hardware-Inventory Test Procedures
**CSF Subcategory:** ID.AM-01 - Inventories of hardware managed by the organization are maintained.

**Scope:** Alma Security 2025 CSF Assessment

**Auditor:** OG <og@almasecurity.com>

---
## Test Procedures

### Review Hardware Inventory 
- Obtain the most recent hardware inventory list from the IT department or asset management system.
- Use network scanning tools (e.g., Nmap) to discover all connected devices on the network.
- Cross-reference the scanned results with the documented inventory.

---

## Evidence Requests
- Current hardware inventory list
- Network scan reports

---

## Related Artifacts
- Hardware Inventory Spreadsheet

## Notes
This test procedure addresses the IDENTIFY function to ensure that your organization's hardware inventory is both accounted for and accurate, aligning with the requirements of ID.AM-01.
